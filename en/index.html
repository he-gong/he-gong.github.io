Posted on 2018-10-17

Step 1：Build a demo locally in your computer

1. Install Git and Node.js

Installing Git and Node.js is easy. You can refers to their official sites and other tutorial. Download Git and Download Node.js
There are options when you install git and I recommend you to select that “use common command in Linux” because that’s more convenient.
After installation，you can click the right button of your mouse anywhere, and there will be a new option of git bash here. This is a tool the same as Linux. It is really convenient. Please enter the directory of your project and open this bash. All the Commands in this article will be executed here. (Please notice: command is different from code. command is executed in bash and code is written in files)
Next, you should install Node.js，Its package manager is called npm, which is used to install packages of Node.js.
Actually Hexo is a package of Node.js，so you have to install Node.js first.

Entering the folder of themes, execute the command of git clone https://github.com/iissnan/hexo-theme-next then you can clone this theme to your computer.
Then change a line containing theme in _config.yml to theme: hexo-theme-next
If you have generated the folder of public，then you need to execute hexo clean to clear all the history files so that the new theme will take effect.
Then we can execute hexo g to generate the folder of public

Step 2：Upload your website to github（Select one method among the two methods）

注意：无论使用哪个方案，你提交修改到github以后，可能无法立刻看到页面的变化，这是因为浏览器中有旧页面的缓存，清除缓存即可，以chrome浏览器为例，勾选more tools>clear browsing data>Cached images and files>clear data即可

方案1：用hexo d来部署网站，这是官方推荐的方式，建议用这个方法

1. 修改配置文件，打开_config.yml,修改最下面的

2.Create the local key，and paste it into your github account

执行以后，密钥创建后默认会保存在C:\user\你的用户名\.ssh中，复制id_rsa.pub文件的内容
进入github，点头像>Setting>SSH and GPG keys>New SSH key，将复制的内容粘贴进去

3. Deploy the website to github

Execute hexo d -g then you can generate the website and deploy it.
