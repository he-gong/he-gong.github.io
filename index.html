第一步：在本地搭建你的个人网站

1. 安装Git和Node.js

安装git的时候选择用Linux的常见命令，这样会比较方便。
安装以后，单击鼠标右键，出现git bash here，这是一个类似Linux下的bash，很方便。后面的操作都是在这里完成，不再赘述。
接下来安装Node.js，它的包安装管理器是npm，就像python的pip、ubuntu的dpkg一样。
Hexo其实是Node.js的一个包，所以必须先装Node.js。

2. 安装Hexo

3. 在本地新建Hexo项目

4. 修改主题：

根据知乎上的探讨有哪些好看的 Hexo 主题？，我得知 https://github.com/iissnan/hexo-theme-next是最常用的主题，我们以它为例。进入themes文件夹，在命令行下输入git clone https://github.com/iissnan/hexo-theme-next即可创建这个主题的文件夹。
然后修改_config.yml的主题那行为 theme: hexo-theme-next
如果已经有public文件夹，那么执行hexo clean清除public整个文件夹，清除以前主题的网页。
然后执行hexo g重新生成public文件夹

第二步：将你的网站上传到github（两个方案选其一即可）

方案1：用hexo d来部署网站，这是官方推荐的方式，建议用这个方法

1. 修改配置文件，打开_config.yml,修改最下面的

2.创建本地密钥，并粘贴到github账户中

执行以后，密钥创建后默认会保存在C:\user\你的用户名\.ssh中，复制id_rsa.pub文件的内容
进入github，点头像>Setting>SSH and GPG keys>New SSH key，将复制的内容粘贴进去

3. 将网站发布到github上

执行 hexo d -g 即生成(generate)并部署(deploy)

方案2：用git来上传网站，优点是不需要搞密钥那些东西，缺点是，占空间多，而且不是官方推荐的方法

1. 在github新建一个项目并下载下来

新建一个项目，命名为username.github.io，其中username指的是你的用户名，必须这样命名才能用。
将这个新建的工程拷贝到.deploy/he-gong.github.io文件夹中，作为备份，这个文件夹跟github上的内容一样，便于我们查看。
git clone https://github.com/he-gong/he-gong.github.io.git .deploy/he-gong.github.io
